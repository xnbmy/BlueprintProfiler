蓝图分析器

一个全面的虚幻引擎蓝图性能分析和代码检查工具。

特色

1. 运行时分析器


实时蓝图节点执行跟踪、
性能热点识别、
执行频率分析、
自动 PIE（编辑器中播放）集成

2. 静态分析（Linter）

死代码检测、
孤立节点识别、
未使用的变量和函数检测、
转换滥用检测

3. 引用分析仪

资产引用分析、
资产使用跟踪、
参考计数可视化

4. 数据导出

CSV 导出支持、
JSON 导出支持、
会话保存/加载功能、
支持的引擎版本

虚幻引擎 5.5

虚幻引擎 5.6

虚幻引擎 5.7

支持平台

Windows 64 位、
macOS、
Linux

安装

把 BlueprintProfiler 文件夹复制到你项目的插件目录

重启虚幻引擎

在 Blueprint Profiler 的 Edit > 插件>Blueprint Profiler>启用该插件

用法

从“Blueprint Profiler”窗口打开>Blueprint Profiler 窗口

使用运行时分析器标签来记录和分析蓝图性能

使用静态分析标签来扫描代码问题

使用内存分析器标签来分析资产引用
