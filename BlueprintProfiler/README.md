# Blueprint Profiler

A comprehensive performance analysis and code inspection tool for Unreal Engine Blueprints.

## Features

### 1. Runtime Profiler
- Real-time Blueprint node execution tracking
- Performance hotspot identification
- Execution frequency analysis
- Automatic PIE (Play In Editor) integration

### 2. Static Analysis (Linter)
- Dead code detection
- Orphan node identification
- Unused variable and function detection
- Cast abuse detection
- Tick complexity analysis

### 3. Memory Analyzer
- Asset reference analysis
- Memory usage tracking
- Reference count visualization

### 4. Data Export
- CSV export support
- JSON export support
- Session save/load functionality

## Supported Engine Versions
- Unreal Engine 5.5
- Unreal Engine 5.6
- Unreal Engine 5.7

## Supported Platforms
- Windows 64-bit
- macOS
- Linux

## Installation

1. Copy the `BlueprintProfiler` folder to your project's `Plugins` directory
2. Restart Unreal Engine
3. Enable the plugin in Edit > Plugins > Performance > Blueprint Profiler

## Usage

1. Open the Blueprint Profiler window from Window > Blueprint Profiler
2. Use the Runtime Profiler tab to record and analyze Blueprint performance
3. Use the Static Analysis tab to scan for code issues
4. Use the Memory Analyzer tab to analyze asset references
